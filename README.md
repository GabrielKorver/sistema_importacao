Este projeto envolve um sistema de login, importação de dados e controle de sessão, com validação em duas áreas principais. A função de login verifica se as credenciais são válidas, armazenando o estado de login no localStorage e redirecionando o usuário para a página do sistema, enquanto o logout remove o estado de login e redireciona para a página de login. A função de verificação de login assegura que o usuário esteja autenticado ao acessar a página do sistema. Na importação de dados, valida-se o rastreio (se não estiver vazio e com o tamanho correto), o cliente (não vazio) e se o rastreio já foi importado anteriormente. Caso todas as validações sejam bem-sucedidas, o rastreio é adicionado ao localStorage e exibido na tabela, com a possibilidade de remoção do item. O localStorage garante a persistência dos dados entre as sessões.
